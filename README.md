# 🌟 AI Image Caption Generator  

> An AI-powered system that automatically generates captions for images using modern tools like ChatGPT Vision and Google Teachable Machine.

---

## 🚀 Overview  
This project focuses on building a simple yet powerful pipeline that takes an image as input and generates meaningful captions in different styles.  

The goal is to make images more **accessible**, **searchable**, and **understandable** using AI.  

---

## 🧠 How It Works  

User Upload → Google Forms → Google Drive → AI Analysis → Image Classification → Caption Generation → Google Sheets

---

## 🎯 Objectives  
- Automate image caption generation  
- Understand basic AI + Computer Vision concepts  
- Implement image classification using no-code tools  
- Build a real-world AI workflow  

---

## 🛠️ Tech Stack  

| Tool | Purpose |
|------|--------|
| ChatGPT Vision / Gemini | Image understanding |
| Google Forms | Image input |
| Google Drive | Image storage |
| Google Sheets | Data storage |
| Google Teachable Machine | Image classification |
| Canva | Diagrams & visuals |

---

## ⚙️ Features  

- 📸 Image Upload System  
- 🤖 AI-based Image Analysis  
- 🏷️ Image Classification (Indoor / Outdoor / Academic / Social)  
- ✍️ Multiple Caption Styles  
  - Descriptive  
  - Creative  
  - Accessibility-friendly  
- 📊 Caption Storage & History  
- 🔍 Search & Filtering (planned)  
- 🔁 Feedback System (planned)

## 📊 Sample Output  

**Input:** Students studying in a library
